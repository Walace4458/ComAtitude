# ComAtitude

Aplicativo mobile desenvolvido em React Native com foco em aprendizado e prática
de desenvolvimento de aplicativos.

## Funcionalidades
- Navegação entre telas
- Interface desenvolvida pelo autor
- Estruturação de componentes
- Lógica básica de funcionamento do app

## Tecnologias
- React Native
- JavaScript
- HTML / CSS (conceitos aplicados à interface)

## Status do projeto
Projeto em desenvolvimento e evolução contínua.
